🐶 Dog API

API simples de imagens de cachorros feita com Node.js + Express

📚 Sobre

Esta API retorna imagens de cachorros em formato JSON. As imagens estão armazenadas no servidor dentro da pasta data/fotos.

🔗 Endpoints

GET /api/cachorros/aleatorio
Retorna uma imagem aleatória de cachorro.
GET /api/cachorros/:raca
Exemplo: /api/cachorros/husky

📦 Exemplo de resposta

{
  "status": "success",
  "message": "http://localhost:3000/fotos/husky/1.jpg"
}

📁 Acesso direto às imagens

/fotos/nome-da-imagem.jpg
